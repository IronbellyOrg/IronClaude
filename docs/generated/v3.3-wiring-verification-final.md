## 5. Reachability Analysis

The AST call-chain reachability analyzer (`reachability.py`) implements FR-4.2:

- **Algorithm**: BFS from entry-point functions through per-module call graphs
- **Cross-module resolution**: Handles `import X`, `from X import Y`, relative imports
- **Lazy imports**: Included in call graph (runtime paths for circular-dependency avoidance)
- **Documented limitations** (NFR-6): Dynamic dispatch (`getattr()`), string-based registry lookups, and `**kwargs` delegation are invisible to static AST analysis
