# Tasklist Index Template

> **Usage**: This template defines the output structure for the tasklist index file
> generated by `/sc:tasklist`. All `{{SC_PLACEHOLDER:*}}` sentinels must be replaced
> during generation. Phase files are emitted separately using the phase template.
>
> **Sentinel self-check**: After generation, verify zero remaining sentinels:
> `grep -c '{{SC_PLACEHOLDER:' <output-file>` should return 0.
>
> **File emission**: The index is one of N+1 files in the tasklist bundle
> (1 index + N phase files). It contains all cross-phase metadata, registries,
> traceability, and templates. Phase files contain ONLY tasks for that phase.

---

```yaml
---
title: "{{SC_PLACEHOLDER:sprint_tasklist_title}}"
roadmap_source: "{{SC_PLACEHOLDER:roadmap_filename}}"
```
